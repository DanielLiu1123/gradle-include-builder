# Configuration Options

This is all the configuration options that can be used in the `include-builder.json` file.

```yaml title="include-builder.json"
{
  "dependencies": [
    {
      "path": "third-party/httpexchange-spring-boot-starter",
      "git": {
        "url": "https://github.com/DanielLiu1123/httpexchange-spring-boot-starter",
        "branch": "main"
      }
    },
    {
      "path": "third-party/grpc-starter",
      "git": {
        "url": "https://github.com/DanielLiu1123/grpc-starter",
        "branch": "main"
      }
    }
  ]
}
```

## JSON Schema

JSON schema is available at [schema.json](https://github.com/DanielLiu1123/gradle-include-builder/blob/main/schema.json).
